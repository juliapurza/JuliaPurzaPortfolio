LAURIER BRANTFORD CAMPUS MAP


I created a Laurier Brantford campus map for my graphic design course using an existing map. This was done to create a better and more optimized quality version, possible for Laurier to make use of in the future.

Below you can see the template and under that, the version I created using Adobe Illustrator. The idea here was to work on mastering graphic design techniques and learning the tools within Illustrator to create an accurate version of the original.
